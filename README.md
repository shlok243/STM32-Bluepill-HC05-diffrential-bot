# STM32-Bluepill-HC05-diffrential-bot
This repo will be based on devlopment of a diffrential based robot which can be controlled using a HC05 bluetooth module. It also has a lcd display to display the action which the robot is performing

Connections - 
PA10 - USART1_RX
PA9 - USART1_TX
PB0 - Motor_1_positive 
PB1 - Motor_1_negetive
PB10 - Motor_2_positive 
PB11 - Motor_2_negetive
PA0 - LCD PIN4 
PA1 - LCD PIN6
PA2 - LCD PIN11
PA3 - LCD PIN12
PA4 - LCD PIN13
PA5 - LCD PIN14
GND - LCD PIN1 
VDD - LCD PIN2
Potentiometer - LCD PIN3

Refer the ioc file for detail config of connections.
Motor driver used - L398N3D
Motors used - N20 motors
